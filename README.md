## Run locally

1. Clone this repo
2. Install **Python 3.12** and dependencies:

```bash
py -3.12 -m pip install pygame pytmx
```

3. Run from the **project root** (not inside `code/`):

```bash
cd jv-pirates-temp
py -3.12 code/main.py
```

Controls: arrow keys to move, Space to attack, P to pause.
